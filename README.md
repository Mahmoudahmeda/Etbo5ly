# Etbo5ly: Food Planner Application

Etbo5ly is an Android mobile application designed to help users discover global recipes and plan their weekly meals. The application integrates with external APIs and cloud services to provide a seamless cooking and organization experience.

## Core Features

* **Recipe Discovery**: Users can explore meals by country of origin, browse specific meal categories, or search by ingredient.
* **Meal of the Day**: A dashboard feature that displays a random meal daily to provide user inspiration.
* **Multimedia Instructions**: Detailed meal views including ingredient lists with images and embedded YouTube video tutorials.
* **Offline Accessibility**: Users can save favorite meals for viewing without network connectivity.
* **Data Synchronization**: Secure backup and restoration of user data (Favorite meals) via Firebase Cloud.
* **User Authentication**: Support for secure login/sign-up and social networking authentication.
* **Guest Access**: A limited navigation mode allowing users to browse recipes without account registration.


## Technical Specifications

The project is developed using modern Android standards and libraries:

* **Language**: Kotlin.
* **UI Framework**: Jetpack Compose following Material Design principles.
* **Architecture**: MVVM pattern.
* **Networking**: Retrofit for communication with The MealDB API.
* **Local Persistence**: Room Database for offline storage and Favorites management.
* **Reactive Programming**: Kotlin Coroutines and Flow.
* **Cloud Integration**: Firebase for Authentication and Data Synchronization.

## Project Structure

* **Splash Screen**: Initial animated entry using the Lottie library.
* **Search Interface**: Grid-based browsing for categories and countries with real-time filtering.
* **Meal Details**: Expandable recipe instructions and interactive favorite/calendar buttons.
* **Profile Management**: User settings including logout functionality and navigation to planned/favorite meals.
